Cameras

Camera is an abstract class

  1. ArrayCamera render the scene from multiple cameras on specific areas of render
    • E.g. old multiplayer games where the screen is split
  2. StereoCamera render the scene through two cameras that mimic the eyes to create a parallax effect
    • E.g. use with devices like VR headset, red and blue glasses or cardboard
  3. CubeCamera do 6 renders, each one facing a different direction. Can render the surrounding for things like environment map, reflection or shadow map
  4. OrthographicCamera render the scene without perspective
      E.g. Age of Empire, size of characters is always the same
  5. PerspectiveCamera render the scene with perspective
